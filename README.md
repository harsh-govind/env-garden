# env-garden

Store and share environment variables securely.

env-garden is an open-source project focused on making `.env` collaboration safer and easier for teams. Instead of passing secrets around in chats or docs, env-garden aims to provide a secure, auditable way to manage and share environment configurations.

## Why env-garden?

- Keep secrets out of messages and screenshots.
- Share environment sets with teammates in a controlled way.
- Reduce setup friction for local development and onboarding.
- Build in the open with transparent security and community feedback.

## Project Status

This project is in early development.

Core functionality and security hardening are actively being built. Expect frequent updates and improvements.

## Getting Started

### Prerequisites

- Node.js 20+
- pnpm (recommended)

### Run Locally

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

### Other Scripts

```bash
pnpm build
pnpm start
pnpm lint
```

## Security

Security is the foundation of env-garden.

If you discover a vulnerability, please do not open a public issue with exploit details. Report it privately to the maintainers first (add a dedicated contact channel here).
